*프로그래밍
=>프로그램을 만드는일 
=>코드입력 -> 번역 -> 실행
번역은 컴파일(compile)이라고 하며, 이를 작업하는 프로그램을 컴파일러 라고함
    -> 컴파일이 잘못되었다 -> 번역이 잘못된거라고 할 수 있다
=>고급 언어: 사람이 이해하기 쉬운 언어
    -> 파이썬이나 스크립트가 해당됨
=>저급 언어: 컴퓨터가 이해하기 쉬운언어
    -> c언어가 해당됨 포인터를 이용해 메모리에 다이렉트로 저장 (중간단계)
=>기계어: 0과 1로만 표현됨


*파이썬
=>문법이 단순하여 초보자가 배우기 쉬움

=>다양한 운영체제에서 사용 가능
=>속도가 느린 단점이 있음
